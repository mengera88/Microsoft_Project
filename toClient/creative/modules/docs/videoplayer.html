Video Player Module

Basic usage

First, use the zepto.min.js file included with this plugin. Then just add the script <script src="js/ut_videoplayer.js"></script> and pass a DOM element to the ut_videoplayer() function! (Note that the icon fonts used for the video player controls will only appear if you're serving the ad via localhost or other means, due to cross-domain origin request limitations.)


      <div id="video_container"></div>
      <script>
        var vp = $('#video_container').ut_videoplayer({
          source_webm: "http://media.w3.org/2010/05/sintel/trailer.webm",
          source_mp4: "http://media.w3.org/2010/05/sintel/trailer.mp4",
          slate: "../img/begin_slate1.jpg"
          endSlate: "../img/end_slate1.jpg"
      });
      </script>

Configuration

You've got some configuration options that are set as properties of a config object passed as a parameter of the ut_videoplayer(config) function. These are the defaults: 


      <div id="video_container"></div>
      <script>
        var config = {
          source_webm: "http://media.w3.org/2010/05/sintel/trailer.webm", // required
          source_mp4: "http://media.w3.org/2010/05/sintel/trailer.mp4", // required
          slate: "slates/begin_slate1.jpg", // required - image shown in the player before play (defaults to black)
          endSlate: "slates/end_slate1.jpg", // required - image shown in the player after completion (defaults to black, or the slate if it is defined)
          displayHidden: false, // (bool) initialize a hidden player
          aspectRatio: "16:9", // width will default to 100% of its container, and the height will be set according to this ratio,
          width: 640, // (int) width in pixels - overrides aspectRatio
          height: 360, // (int) height in pixels - overrides aspectRatio
          muted: false, // (bool) initialize the player in a muted state,
          volume: 0.75, // (0-1)
          clickID: 1, // (int) clicks redirect to the corresponding click
        };

        var vp = $('#video_container').ut_videoplayer(config);
      </script>

Methods

After storing a reference to the object returned by ut_videoplayer(), you can then call the following methods on it:

  • vp.play() should be bound to a user interaction, as autoplay is not supported across browsers and devices.
  • vp.pause()
  • vp.paused() returns boolean
  • vp.showPlayer()
  • vp.hidePlayer()
  • vp.mute(boolean) toggle mute status.
  • vp.clickable(boolean) toggle clickability of all video player elements.
  • vp.isClickable() returns boolean.
