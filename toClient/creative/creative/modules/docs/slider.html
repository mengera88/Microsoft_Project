This is UT Slider

You can put nearly any kind of content inside as many divs as you'd like.

We'll generate the bullets for you.
And start the sliding automatically.

$('#slider').utslider(options);
The options argument is optional. 
default_options = {
  autostart: true,        // start sliding automatically?
  interval: 5000,         // how long between slides
  delay: 5000,            // how long to wait before resuming
  speed: 600,             // how quickly  the slides slide
  effect: 'ease',         // sliding animation effect
  makebullets: true,      // generate clickable bullets
  makepagebuttons: true,  // generate left and right buttons
  fade: true,             // fade slides as they slide in and out
  autoheight: true,       // set slider height based on contents
  autovalign: true,       // vertically center each slide
  rtl: false,             // slide from right to left
  pauseable: true         // generate pause/play toggle button
}
